Aplicaciones Web

¿Qué es AW?

La asignatura de Aplicaciones Web se centra en el desarrollo y diseño de aplicaciones web utilizando tecnologías como HTML y CSS. Los estudiantes aprenderán a crear interfaces de usuario atractivas y funcionales. Además, se explorarán plataformas populares como WordPress y Moodle para la creación y gestión de sitios web. Esta asignatura proporciona una base sólida para aquellos interesados en el campo del desarrollo web, preparando a los estudiantes para enfrentar los desafíos del mundo real en la creación de aplicaciones web modernas.
